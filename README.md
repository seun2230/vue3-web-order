## 🔥WEB-ORDER-SYSTEM.v1 기능
### 관리자 페이지(Admin page)    

- **상품 관리**
  1. 상품 메뉴 폼 추가 및 이미지 업로드
  2. 상품 메뉴 관리 테이블 추가, 삭제
  3. 상품 주문 내역 관리 및 조리 완료 내역 기능   

- **사용자 관리**
  1. 총 리뷰 게시물 관리 테이블 추가, 삭제
  2. 가입한 사용자 관리 테이블 추가, 삭제   

- **홈화면 배너 이미지 관리** 

- **슬라이드 이미지 & 공지 관리** 
  - 게시물 이미지 없는 경우 기본 이미지 추가,수정, 삭제

### 사용자 페이지(User page)   

- **회원관리 기능** 
1. 회원가입 시 사용자 정보 입력 
2. 마이페이지
    - 사용자 회원 정보 수정 추가
    - 메뉴 평가를 위한 리뷰 작성 추가
    - 리뷰 좋아요 한 게시물 리스트 확인
3. 폼검증 기능    

- **메뉴 기능** 
  1. 상품 수량 조절, 메뉴 추가, 선택 삭제
  2. 메뉴 선택과 주문 결제 선택을 위한 스크롤 UI 추가
  3. 주문 상세 내역 기능  
<br />

- **리뷰 기능** 
  1. SNS형 구조를 위한 시간순으로 리뷰 정렬
  2. 개인정보수집
    - 주문한 고객과 정보 수집에 동의한 사용자만 리뷰 반영
  3. 주문한 사용자의 리뷰 작성, 수정, 삭제 추가
  4. 리뷰 작성 시 신뢰성 향상을 위한 평점, 키워드 추가
  5. 리뷰 상세 페이지의 댓글, 좋아요 등 소셜 기능 추가
